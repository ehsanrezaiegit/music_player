## ⚡ Features

- 🎵 Play music files from device storage
- ▶️ Play, Pause, Stop controls
- ⏭️ Next & Previous track
- 🔀 Shuffle mode
- 🔁 Repeat (single / all)
- 📋 Playlist management
- 🔍 Search songs
- 🎨 Beautiful and modern UI
- 📱 Smooth and responsive design
- 🔔 Notification controls
- 🎚️ Volume control
- advancement bar / Seekbar
- 📁 Browse music by folders
- 🖼️ Album art display

> ⚠️ Remove features you don't have, 
> add features specific to your app
